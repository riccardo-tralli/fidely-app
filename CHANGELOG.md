# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),  
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-23

### Added
- Add card creation with customization options
- Add card search to quickly find saved loyalty cards
- Add favorites to pin frequently used cards
- Add light and dark theme modes
- Add barcode scanning support
- Add card photo attachments
- Add multi-language support (EN, ES, FR, IT)
- Add multiple card display modes
